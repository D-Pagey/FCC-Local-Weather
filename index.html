Local Weather App

This is an Intermediate Front End Developer project from the FreeCodeCamp curriculum. The user stories to be completed are:

  • I can see the weather in my current location.
  • I can see a different icon or background image (e.g. snowy mountain, hot desert) depending on the weather.
  • I can push a button to toggle between Fahrenheit and Celsius.

This was made using: Flexbox, SCSS, and native JS. No CSS or JavaScript frameworks or libraries were used. The code can be seen on Github.
